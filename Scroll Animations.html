Stagger Fade-Up

Cards that arrive
one after another.

Each child element in a .stagger container receives an incremental transition-delay, creating a cascade effect.

Fast by default

IntersectionObserver fires once per element, with zero scroll-event overhead.

Fully configurable

Set threshold, root margin and delay per element with data attributes.

Accessible

Respects prefers-reduced-motion -- animations disable cleanly.

Zero dependencies

Pure HTML, CSS and one small IntersectionObserver. Ship it anywhere.
